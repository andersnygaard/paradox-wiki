# Paradox Wiki

Knowledge distilled from the **Paradox** builders' mailing-list archive (Yahoo
Groups `paradoxbuilders` / `paradoxbuilders2`, later migrated to groups.io
`paradoxsailboat`). The Paradox is a small plywood microcruiser sailboat
designed by Matt Layden, sailed single-handed from within an enclosed cabin.

This wiki is grown incrementally: each pass reads a batch of archived emails and
contributes new, sourced knowledge. Recurring, well-supported themes are
promoted toward the root ("trunk"); one-off details live deeper in the tree.
