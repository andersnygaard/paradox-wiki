# Components — browse the Paradox by part

This index mirrors the physical-part categories used to organise the project's build-photo library, so the knowledge tree can be navigated the same way as the pictures. Each entry links to a **component hub**: a short description of the part plus links to where it is covered in the tree.

The thematic branches (`design/`, `construction/`, `operations/`) remain the current home of most detailed content; each component hub routes to it, and is intended to grow into the canonical home for that part as content is consolidated. Parts still awaiting their own detailed treatment are marked *(content to come)*. Build *steps/techniques* live under `construction/`, rig/hull *alternatives* under `design/variants/`, and individual built hulls under `boats/` — see the footer.
